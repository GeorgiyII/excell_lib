Программа обрабатывает страницу excel документа с таблицей расчета гипсовых стен, имеющюю структуру как показано на изображении ниже, и добавляет столбцы с материалами стен и формулами расчета цены.

Для использования программы необходимо создать строку с шифрами материалов стен.

Необходимо создать страницу с таблицей где каждый шифр материала имеет полное название и цену, которая в последствии будет подвязываться к новым колонкам.

В результате работы программы создается новая страница в файле Excel, в связи с этим можно не волноватся о потере данных рабочей таблицы.

Программа принимает только Excel файлы имеющую структуру описаную выше, любые другие файлы обрабатыватся не будут, а таблицы с несоответствующей структурой получат лишь копирование указанной страницы Excel.
